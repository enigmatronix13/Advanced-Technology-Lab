# Patient Dashboard

Create a patient profile creation and appointment booking, built with React Native, Expo, and TypeScript.

## Tech Stack

- React Native + Expo
- TypeScript
- expo-router (file-based routing)
- react-native-reanimated, react-native-gesture-handler
- React Native Web

## Modules

**Create Profile** — Collects patient name, phone, and age with input validation. Errors surface as alerts; successful submissions auto-reset the form after 3 seconds.

**Book Appointment** — Physician and time slot selection via native pickers. Confirmation auto-dismisses after 3 seconds.
